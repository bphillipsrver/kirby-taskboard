## Implementation Instructions

### For WordPress (Kadence Theme)

**Step 1: Homepage Title**
1. Go to WordPress Admin ‚Üí Pages ‚Üí Home
2. Edit page
3. Look for "SEO" or "Page Title" section (may be in Kadence settings)
4. Replace title with recommended option
5. Save as draft (don't publish yet)

**Step 2: Meta Description**
1. Install Yoast SEO plugin if not already installed
2. Edit homepage
3. Scroll to "Yoast SEO" box at bottom
4. Enter meta description in "Meta description" field
5. Save as draft

**Step 3: Review & Approve**
- Preview changes
- Confirm titles display correctly in browser tab
- Confirm meta description length (Yoast shows green/orange/red indicator)

**Step 4: Publish**
- Once approved, publish changes
- Submit updated sitemap to Google Search Console (covered in separate task)
